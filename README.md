## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 16
- [npm](https://npmjs.com/) >= 8
- Supported OSes: Linux, macOS and Windows


### Installation

```sh
npm run build
```

### Usage
```sh
# Check the setup went well
npm run check

# Run
npm start
# Go to http://localhost:1337

```
